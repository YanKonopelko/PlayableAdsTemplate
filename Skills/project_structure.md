# Структура проекта (PlayableAdsPVZ)

Проект представляет собой Playable Ad (играбельную рекламу) на базе **Cocos Creator 3.8.7**.

## Основная структура папок `assets/`
- `Animations/` — Файлы анимаций (AnimationClip, AnimationController).
- `Audio/` — Исходные аудиофайлы (звуки, музыка).
- `Fonts/` — Шрифты, используемые в UI.
- `Materials/` — Пользовательские материалы для рендеринга.
- `MyBuild/` — Вспомогательные файлы или конфигурации для специфичных сборок.
- `Prefabs/` — Префабы (готовые к инстанцированию узлы с компонентами).
- `Scenes/` — Игровые сцены (содержит основную сцену).
- `Scripts/` — Исходный код игры (TypeScript).
- `Sprites/` — 2D текстуры и атласы для UI и игровых объектов.

## Структура кода `assets/Scripts/`
- `Extensions/` — Расширения базовых классов или методов.
- `Sounds/` — Логика управления звуками и аудио контроллеры.
- `Utills/` — Утилитарные классы и хелперы (например, для математики, логирования, событий).
- `GameplayScene.ts` — Основной скрипт управления геймплеем и потоком играбельной рекламы (включая логику перехода в стор).
